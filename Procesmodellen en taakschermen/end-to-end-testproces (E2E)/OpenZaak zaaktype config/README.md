# Zaaktype importeren

1. Ga naar de beheerinterface van OpenZaak
2. Ga naar Catalogi
3. Open de gewenste catalogus
4. Klik onderaan de pagina op de button "Zaaktype importeren"
5. Selecteer het ZIP-bestand met daarin de zaaktypeconfiguratie

⚠️ Let op! Houd er rekening mee dat er met het importeren van zaaktypes enige vervuiling kan ontstaan in jouw zaaktypecatalogus.
