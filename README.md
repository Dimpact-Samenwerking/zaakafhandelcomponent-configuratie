# BPMN processen in ZAC

Binnen de [zaakafhandelcomponent](https://github.com/infonl/dimpact-zaakafhandelcomponent) (ZAC) kunnen BPMN processen worden gebruikt om specifieke (gemeentelijke) werkprocessen uit te voeren.  
Om deze functionaliteit te kunnen testen hebben verschillende gemeenten een bestaand proces nagebouwd. Dat is gedaan met een combinatie van [Flowable](https://www.flowable.com/) voor het proces, en [Form.io](https://form.io/) voor de schermen.

Een eerste versie van deze processen en bijbehorende schermen is te vinden onder [Procesmodellen en taakschermen](./Procesmodellen%20en%20taakschermen).
Daarnaast wordt er gewerkt aan een set voorbeeldcomponenten. Deze kunnen gebruikt worden om (met beperkte JSON-kennis) taakschermen te maken. Deze zijn te vinden onder [Taakscherm snippets](./Taakscherm%20snippets).
